## Funcionalidades

- Sistema de login simples (usuário/senha)
- Autenticação e validação via JWT
- Registro de usuários (Nome, Sobrenome, E-mail, etc.)
- CRUD de pedidos de oração
- Soft delete de produtos (registro lógico)
- Associação dos pedidos ao ID do usuário
- Painel administrativo para visualizar e editar pedidos
- Link de encaminhamento para doação
